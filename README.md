# МУР&ГАВ — интернет-магазин зоотоваров (демо)

Демонстрационный прототип интернет-магазина зоотоваров с программой лояльности «Хвостик».
Прототип на HTML/CSS/JS (мини-фреймворк в `support.js`): каталог, карточка товара, корзина,
оформление заказа, авторизация и личный кабинет с бонусным счётом.

## Запуск локально

```bash
python -m http.server 8000
# открыть http://localhost:8000/
```

## Структура

- `index.html` — приложение (все экраны)
- `support.js` — рантайм прототипа
- `images/` — фото товаров и обложка

> Это демонстрационный прототип, не продакшн-код.
